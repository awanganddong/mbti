# 页面设计文档：首页卡片列表与测试页（桌面端优先）

## 全局设计（适用于所有页面）

### Layout
- 桌面端优先：内容最大宽度 1200px，居中；左右留白随屏幕增大。
- 首页卡片区使用 CSS Grid：`grid-template-columns: repeat(3, minmax(0, 1fr))`；在窄屏降为 2 列/1 列。
- 间距体系：8px 基础单位（8/16/24/32）。

### Meta Information
- 默认 Title 模式：`{页面名} - MBTI`。
- 默认 Description：一句话说明“性格测试入口与结果查看”。
- Open Graph：`og:title/og:description/og:type=website`（如未配置分享图，可省略 `og:image`）。

### Global Styles
- 背景：`#0B1220`（深色）或 `#FFFFFF`（浅色，二选一，保持统一）。
- 主色（Accent）：`#6366F1`（Indigo）。
- 字体：系统字体栈；标题层级 24/20/16；正文 14/16。
- 卡片：圆角 12px，阴影轻量，hover 提升阴影与边框高亮。
- 按钮：主按钮填充色 + hover 变深；次按钮描边。
- 链接：主色文本，hover 下划线。

## 页面 1：首页（测试卡片列表）

### Page Structure
- 顶部：页面标题区（左对齐）。
- 主体：测试卡片 Grid。

### Sections & Components

#### 1) Header（页面标题区）
- 元素：
  - H1：`测试中心`/`选择一个测试开始`（二选一）。
  - 简短说明：例如“从下方卡片选择你要做的测试”。

#### 2) TestCardGrid（卡片列表）
- Grid 布局：
  - Desktop：3 列；卡片高度自适应。
  - 卡片间距：24。
- 渲染方式：数据驱动（遍历 `TestCatalogCard[]`）。

#### 3) TestCard（测试卡片：以“MBTI 测试”为例）
- 卡片结构（从上到下）：
  1. 顶部区域：
     - 标题（必填）：MBTI 测试
     - 简介/副标题（可选）：一句话描述
     - 标签（可选）：如“性格”“经典”
  2. 子卡片列表区域（必填）：
     - 纵向堆叠或小型列表（建议 2~4 个可视，更多可滚动/展开，按需实现）。

#### 4) SubTestCard（MBTI 子卡片）
- 视觉：浅底/描边块 + 右侧箭头图标（表示可点击）。
- 子卡片结构：
  - 左侧：
    - 子卡片标题（必填）：如“MBTI（93题）”
    - 子卡片描述（可选）：如“更精细结果”
  - 右侧：
    - Badge（可选）：如“推荐/新版”
    - 预计耗时（可选）：如“10 分钟”
- 交互：
  - hover：描边高亮、背景轻微变化。
  - click：跳转到 `/test/:testId`（例如 `/test/mbti-93`）。

#### 5) 扩展更多测试卡片（可扩展设计点）
- 新增测试时：直接增加新的 `TestCard`（或在已有卡片中增加新的 `SubTestCard`）。
- 为避免首页拥挤：
  - 卡片内容优先展示 2~3 个子卡片，其余以“更多”行占位（是否实现由后续迭代决定；本次可先不做交互，仅预留结构位）。

## 页面 2：测试页（通用测试容器）

### Page Structure
- 顶部：返回入口 + 测试标题。
- 主体：题目卡片 + 选项。
- 底部：上一题/下一题/提交（按测试定义）。
- 结果区：同页展示结果（完成后替换题目区或在下方展开）。

### Sections & Components

#### 1) TopBar
- 左侧：`← 返回首页`（点击回到 `/`）。
- 中部/左侧：当前测试标题（来自 testId 对应配置）。

#### 2) QuestionCard
- 显示：题干文本。
- 进度：`第 x / n 题`（简洁文本）。

#### 3) OptionsList
- 选项为可点击列表（radio 风格）。
- 选择后高亮当前选项。

#### 4) NavigationActions
- 按钮：上一题（可选）、下一题、提交/完成。

#### 5) ResultPanel
- 完成后展示：
  - 结果标题（如 MBTI 类型）。
  - 简要说明文案（来自配置/计算结果）。
  - 操作：`重新开始`、`返回首页`。

#### 6) 异常/空态
- 若 testId 无对应配置：
  - 文案：`未找到该测试`。
  - 按钮：`返回首页`。
